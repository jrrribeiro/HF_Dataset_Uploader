Gradio demo moved to Temp/gradio_ui

This folder contains the original Gradio-based web UI used for development and demonstrations. It is not packaged into the native EXE.

To run locally:

1. Create a virtualenv and install dependencies from the project `requirements.txt`.
2. From the repository root run:

   python hf_upload_app/Temp/gradio_ui/web_ui.py

Note: The native EXE now uses a Tkinter-based GUI in `hf_upload_app/native_ui.py`.
